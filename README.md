# Cranfield VSM Information Retrieval Project

This repository contains an implementation of a Vector Space Model (VSM) for the Cranfield dataset.

## Files

- `VSM.py`: Main script that preprocesses documents and queries, computes TF-IDF vectors, ranks documents by cosine similarity, and calculates precision/recall metrics.
- `cran.all.1400.txt`: Original Cranfield document collection.
- `cran.qry.txt`: Original Cranfield queries.
- `cranqrel.txt`: Cranfield relevance judgments for evaluation.
- `preprocessed_cranfieldDocs/`: Preprocessed document files created by `VSM.py`.
- `preprocessed_queries.txt`: Preprocessed queries created by `VSM.py`.
- `result.txt`: Final evaluation output generated by `VSM.py` with query-level precision/recall and averages.
- `.venv/`: Python virtual environment for dependency isolation.

## Requirements

- Python 3
- `nltk`
- `pandas`
- `numpy`
- `beautifulsoup4`

## Setup

1. Activate the virtual environment:

   ```powershell
   .\.venv\Scripts\Activate.ps1
   ```

2. Install dependencies if needed:

   ```powershell
   python -m pip install nltk pandas numpy beautifulsoup4
   ```

3. Run the main script:

   ```powershell
   python VSM.py
   ```

4. The metrics output will be saved to `output.txt`.

## Notes

- The script preprocesses text, computes TF-IDF weights, ranks documents using cosine similarity, and evaluates retrieval performance.
- `result.txt` contains precision, recall and fi score values formatted to five decimal places.
